R 서버 사용 주의사항
1. Rserve 라이브러리 다운로드
2. cmd에서 R 실행 
3. library(Rserve) 
4. Rserve(FALSE,port=6311,args='--RS-encoding utf8 --no-save --slave --encoding utf8  --internet2') 

cmd동작 시 사용가능
